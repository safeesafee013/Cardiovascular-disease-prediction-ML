# Cardiovascular Disease Prediction using Machine Learning

This project implements a machine learning model to predict the likelihood of cardiovascular disease using patient clinical data and health indicators.

## Project Objective
Early detection of cardiovascular disease can help in preventive treatment and reduce mortality. This model predicts the risk of cardiovascular disease based on several medical attributes.
## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Machine Learning Algorithm
## Dataset Features
The model uses clinical health parameters such as:
- Age
- Gender
- Blood Pressure
- Cholesterol
- Glucose Level
- Smoking Status
- Alcohol Intake
- Physical Activity
- Body Mass Index

## Machine Learning Models Used
Some algorithms explored include:
- Logistic Regression
- Random Forest
- Decision Tree
- Support Vector Machine

## Workflow
1. Data preprocessing
2. Data cleaning
3. Feature selection
4. Model training
5. Model evaluation
6. Prediction of cardiovascular disease risk
## Results
The trained model predicts whether a patient is at risk of cardiovascular disease based on clinical parameters.
